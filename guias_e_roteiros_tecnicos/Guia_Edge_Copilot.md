# Edge Copilot - Guia Técnico S122

## 1. Visão Geral (O Básico)
**Para que serve:** Assistente de inteligência artificial integrado ao navegador Microsoft Edge, ideal para suporte em ferramentas web (Wokwi, Node-RED).
**Contexto M3F:** [X] Percepção (Edge) | [X] Aplicação (Cloud)

## 2. Fluxo de Dados (I/O)
- **Entrada (Input):** Texto selecionado no navegador, capturas de tela, prompts de conversação.
- **Saída (Output):** Explicações técnicas, código corrigido, links para documentação.

## 3. Instalação e Configuração

### 🪟 Windows
1.  **Download Edge:** https://www.microsoft.com/pt-br/edge/download
2.  **Procedimento:** Clique no ícone colorido do Copilot no canto superior direito do navegador.
3.  **Login SENAI:** Use sua conta de aluno para habilitar recursos institucionais.

## 4. Integração (Ecossistema S122)
- **Conecta com:** Wokwi (para debug de hardware simulado).
- **Utiliza:** Node-RED e InfluxDB para gerar consultas em linguagem Flux.

**Links Relacionados (CORE):**

## 6. Solução de Problemas (Troubleshooting)
- **Barra lateral sumiu:** Verifique as configurações de aparência do Edge.
- **Copilot não lê o site:** Habilite a permissão "Permitir que a Microsoft acesse o conteúdo da página".
