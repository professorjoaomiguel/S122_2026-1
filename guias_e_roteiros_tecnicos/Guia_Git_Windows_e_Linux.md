**Contexto M3F:** [X] Gestão | [X] Documentação | [X] Colaboração

## 2. Fluxo de Dados (I/O)
- **Entrada (Input):** Código local, commits, pull requests de colegas.
- **Saída (Output):** Repositório remoto atualizado, histórico de versões, documentação (README.md).

## 3. Instalação e Configuração

### 🪟 Windows
1.  **Download:** https://git-scm.com/download/win
2.  **Procedimento:** Escolha "Git Bash" durante a instalação.
3.  **Verificação:** `git --version`

### 🐧 Linux
1.  **Comando de Instalação:** `sudo apt install git`
2.  **Verificação:** `git --version`
